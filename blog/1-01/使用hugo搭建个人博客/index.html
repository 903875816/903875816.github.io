安装hugo

  1. 下载Hugo的安装包。(主要介绍Windows的操作)

    • Windows可以自行下载。下载地址

    • Ubuntu用户，可以使用apt快速安装

      sudo apt-get install hugo

  2. 解压，把hugo.exe文件放到D:\Sofotware\hugo\hugo.exe(路径自定义，但尽量保证是英文路径)。

  3. 把 D:\Sofotware\hugo\加到PATH。

  4. 重启终端，安装完成。

参考： Hugo官方安装使用教程

博客的搭建

  1. 进入Hugo的官方主页,点击快速开始，从第二步开始执行。

  2. 建立新网站

    hugo new site quickstart

  3. 添加主题(默认主题)

    1. cd quickstart
2. git init
3. git submodule add https://github.com/budparrgohugo-theme-ananke.git themes/ananke`
4. echo 'theme = "ananke"' >> config.toml

切换主题

官方教程自带的主题挺丑的，所以我准备换一个，换了一个，还是觉的挺丑的（可能是我不会用，手动二哈），准备再换一个，前面主题遇到的坑一会再说，先开始重新开始换主题

  1. 点击这里去选择主题
  2. 这一主题挺好看的试一下.

  3. 将网页往下翻，有一个Quick Star的标题，里面写的是如何下载主题，但是有点儿问题，这里用下面的代码打开电脑的终端将下面的代码clone到你的电脑本地的themes文件夹内。

    cd theme
git clone https://github.com/orianna-zzo/AllinOne.git themes

  4. 等待其下载完成，进入themes/AllinOne/exampleSite将其的config.toml拷贝出来并删掉原来的config.toml文件

  5. 进入本地的http://localhost:1313/，可以发现主题就切换成功了

配置主题

大部分的配置文件的内容都在config.toml中，我们跟着[官方文档](https://themes.gohugo.io/allinone/#usage)来就行了。

将本地博客上传Github并绑定

将Blog上传到本地仓库

  1. 进入Pbulic
  2. 输入命令git init将文件git初始化
  3. git add .
  4. git commit .将文件上传到本地仓库

上传到github仓库

  1. 在github上建立一个远程仓库，命名为xxx.github.io，xxx一定要是你的username,这个可以点击你的github账号的profile查看
  2. 输入命令get remove add origin git@github.com:xxx/c.github.io.git
  3. git push -u origin master

在github上查看的博客

  1. 进入github的xxx.github.io仓库，点击setting
  2. 把网页往下拉，可以发现一个GitHub Pages选项 Your site is published at http://orangec.info/
  3. 点击网址即可进入
