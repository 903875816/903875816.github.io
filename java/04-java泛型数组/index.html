泛型数组列表(Arraylist)

总览

定义

一个采用类型参数（type parameter）的泛型类（generic class)

格式

使用一对尖括号将类名括起来加在后面作为保存数组列表的对象

Arraylist<Object>
Arraylist<Object> list = new Arraylist();

Arraylist的常用方法

1. add 方法

  • 将对象添加到数组列表的方法

    list.add(new Object("...",))

2. ensureCapacity方法

  • 数组列表空间的分配方法

    list.ensureCapacity(100); Arraylist<Object> list = new Arraylist<>(100);

3. size方法

  • 返回数组列表的实际个数（类似与lenth方法）
  • 当确认数组列表的大小不再发生变化就可以调用trimToSize方法，垃圾回收器将回收多余的储存空间

Arraylist的元素操控

  • 设置元素

    list.set(i,xxxx)

  • 获得元素

    Object e =list.get();

注意(如图)

  • 遍历整个数组
    for(Object element:list) do somethings with e
