一维数组

数组是一种数据结构用来储存同一类型值的类型的集合，通过a[i]可以返问数组中为i的整数

一维数组基础知识

  1. 创建数组
    int[] a = new int[]；int[] xx = {a1,a2,a3}
  2. 赋值方法
    a[i]=i
  3. 初始化
    • 数字数组 所有的元素都初始化为0
    • boolean数组 数组的初始化为false
    • 对象数组 null
  4. 数组的长度
    arrays.length
  5. 命令行参数
    • 其实public static void main(String[] args){}就是一个只接受String类型的数组。

一维数组的基本使用

  1. 遍历所有的元素
    • for 语句
      for(int i=0;i<a.length;i++)
    • for each语句
      for(type variable: collection)
      • 注意：collection必须是数组或者Iterable接口
  2. 所有的元素
    • for each语句和for 语句
    • Arrays.toString 方法
      • 返回一个包含数组元素的字符串，并用逗号分隔[2,3,4,5,6]
  3. 数组的排序
    • 数组可以使用Arrays.sort方法进行排序，其使用的是快速排序的算法，返回值是一个数组

例子说明

    import java.lang.reflect.Array;
    import java.util.Arrays;

    public class array {
        public static void main(String[] args) {
        int[] a={1,22,5,4,7,32,12,76,31,53};
        System.out.println(a);//数组是不能直接打印的


        //想要打印每个元素要用下列方法：
        // for each打印所有的数组
        for (int element:a){
            System.out.print(element+"\t");
        }
        //toString打印所有的数组
        System.out.println("\n" + Arrays.toString(a));//已经变成字符串可以打印
        //数组排序方法
        Arrays.sort(a);//最后还是返回给了int[] a,所以不能使用 System.out.println打印
        System.out.println(Arrays.toString(a));//打印排好序的数组


        //创建一个小的子集
        int[] b = Arrays.copyOf(a,4);//返回的是一个数组的子集,[a0-a4]
        System.out.println(Arrays.toString(b));
        int[] c = Arrays.copyOfRange(a,4,9);//返回的是一个数组的子集,[a5-a10]
        System.out.println(Arrays.toString(c));

        //用二分法查找一个数，并返回这个数的下标
        System.out.println(Arrays.binarySearch(a,7));

        //如果两个数组大小相同，下标元素也相等的话，返回true
        int[] d=a;
        System.out.println(Arrays.equals(a,d));

    }
}
